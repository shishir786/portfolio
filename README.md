## ⚙️ Getting Started

### Clone Repository

```bash
git clone https://github.com/shishir786/portfolio.git
```

### Navigate to Project

```bash
cd portfolio
```

### Install Dependencies

Using PNPM:

```bash
pnpm install
```

or NPM:

```bash
npm install
```

### Run Development Server

```bash
pnpm dev
```

or

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

---

## 🔧 Environment Variables

Create a `.env.local` file:

```env
# Example
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

Refer to `.env.example` for all required variables.

---

## 🚀 Build for Production

```bash
pnpm build
```

Start production server:

```bash
pnpm start
```
